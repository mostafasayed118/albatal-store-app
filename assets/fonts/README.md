# Al Batal Elite font assets

The application embeds **Montserrat** for brand/display typography and **Inter** for product and utility copy. Both are distributed under the SIL Open Font License 1.1; see `OFL.txt`.

- `Montserrat-VariableFont_wght.ttf` — headings, display, and brand wordmark
- `Inter-VariableFont_opsz,wght.ttf` — body, labels, prices, and interface text

The font families are registered in `pubspec.yaml` and applied centrally in `lib/shared/theme/app_theme.dart`.
